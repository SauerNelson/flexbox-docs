Display: Flex

A propriedade display: flex deve ser a primeira a ser declarada no CSS. A partir dela é que o CSS vai entender que precisa estilizar o conteúdo do contêiner com a CAIXA FLEXIVEL

Valores padrão das propriedades do Flexbox
Propriedade Valor
flex-direction row
justify-content flex-start
align-content flex-start

justify-content:

A propriedade justify-content pode ser usada para justificar o conteudo do elemento conteiner em seu eixo primario, ou seja, eixo x (row, row-reverse) ou eixo y (column, column-reverse).

Valores:

center

O valor "center" vai alinhar o conteudo ao centro do eixo primario.

Exemplo:

1

4

space-between

O valor "space-between" vai definir que os itens tenham espaço entre eles.

flex-direction

A propriedade flex-direction define o eixo principal, dando uma direção para os itens flexiveis dentro do container

Valores:

row

Esse valor é o valor padrão da propriedade flex-direction. Os itens serão exibidos horizontalmente, no eixo x, como uma linha.
